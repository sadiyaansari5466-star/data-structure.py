# Data Structures Implementation

This project contains basic implementations of Stack and Queue using Python.

I created this to understand how fundamental data structures work internally instead of just using built-in functions.

## What is included
- Stack (LIFO - Last In First Out)
- Queue (FIFO - First In First Out)

## Concepts used
- Lists in Python
- Basic operations like push, pop, enqueue, dequeue

## Why I made this
Data structures are the base of problem solving in programming. This project helped me understand how data is stored and managed in different ways.

## What I learned
- Difference between stack and queue
- How operations affect data flow
- Importance of choosing the right data structure

## Output
Shows basic operations of stack and queue with example values.
